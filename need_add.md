### Tab 2：条款工作台

对应 **功能四：公共/私有参数分离** + **功能六：RBF条款工作台**

**三区分离架构：**

```
┌─────────────────────────────────────────────────────────┐
│  公共条款区（双方可见，灰显为名义兜底值）                   │
│  融资金额 | 分成比例 | YITO封顶APR | 合作期限              │
├─────────────────────────────────────────────────────────┤
│  私有预测区（仅自己可见）                                 │
│  预测月均营业额 ← 参考值：系统预估 / 融资方预估 / 自行填写  │
├─────────────────────────────────────────────────────────┤
│  派生指标区（仅自己可见）                                 │
│  月回款 | 建议可融资金额上限 | 分成比例 | 回本月数         │
│  YITO触达总回款 | 预估触达月数 | 实际APR | 回收倍数        │
└─────────────────────────────────────────────────────────┘
```

**双向联动公式（三组）：**
- 融资金额倒推：`建议金额 = 月回款 × 触达月数 / (1 + APR × 触达月数 / 12)`
- 分成比例倒推：`建议比例 = 金额 × (1 + APR × 触达月数 / 12) / (月均营业额 × 触达月数)`
- 正推验证：`触达月数 = 金额 / (月回款 - 金额 × APR / 12)`

**只调公共参数 → 点"提交方案"进入谈判**

---

### Tab 3：表达意向

对应 **功能三：阶段三 · 表达意向**

| 子功能 | 说明 |
|--------|------|
| **结构化意向填写** | 投资类型（RBF固定）、金额区间（预设档/自定义）、核心关注点（多选）、备注（可选） |
| **意向摘要确认** | 系统自动生成摘要（项目+金额+关注点+参考区间），确认后发给融资方 |
| **融资方响应** | 融资方"接受沟通" → 进入 Tab 2 正式谈判；"暂不考虑" → 留在列表 |

---

### Tab 4：谈判

对应 **功能七：方案提交与谈判循环** + **功能八** + **功能十**

| 子功能 | 说明 |
|--------|------|
| **方案提交** | 从条款工作台带入公共参数，添加备注后提交 |
| **方案响应** | 对方查看公共参数，用自己私有参数计算派生指标。可：接受 / 拒绝（含理由）/ 反提案 |
| **方案撤回** | 提案方在对方响应前可撤回，保留在时间线（标已撤回） |
| **多方案对比** | 存A/B/C草稿，并排对比公共参数；派生指标各按自己私有参数计算 |
| **条款确认（达成）** | 一方选方案 → 双方各自确认 → 锁定公共条款 → 流转合约通；签署前仍可撤回 |
| **沟通纪要** | 上传线下纪要（文字/文档/图片），逐条确认或有争议循环，确认条款自动同步公共参数 |
| **邀请协作** | 生成邀请链接（7天有效），可指定"谈判者"或"观察者"角色；观察者不可见私有参数 |

---

### Tab 5：时间线

对应 **功能九：全程操作留痕** + **功能十部分**

| 子功能 | 说明 |
|--------|------|
| **谈判时间轴** | 事件：提案 / 接受 / 拒绝 / 撤回 / 纪要上传 / 确认 / 争议 / 补充 / 邀请 |
| **事件字段** | 时间戳、操作人（用户名+角色）、操作类型、操作内容摘要（仅公共参数） |
| **历史版本加载** | 点击任意历史提案 → "基于此版修改"加载到工作台 |

## 内部功能间数据流

| 从 | 到 | 数据 |
|----|----|------|
| Tab 1（做功课） | Tab 2（工作台） | 用户选中的月均营业额偏好 |
| Tab 2（工作台） | Tab 4（谈判） | 公共参数（提交方案） |
| Tab 4（谈判） | Tab 5（时间线） | 提案/接受/拒绝/撤回/纪要事件 |
| Tab 4（谈判） | 合约通 | 条款摘要（公共参数 + 确认时间戳） |
| 合约通 | Tab 4（谈判） | 合同签署状态（判断撤回窗口是否有效） |

---

## 核心原则

**所有输出至合约通的数据仅包含公共参数。私有参数（预测月均营业额）和派生指标（月回款、触达月数等）仅在参与通内部使用，永不向下游传递。**
